Given a csv file with header:
convID,expected,time_to_first_token,token_count
where convID is a string and all other columns are integers
In python, create a scatter plot of time_to_first_token (x-axis) vs token_count (y-axis)
use a red dot to mark the position of each convID that has expected=0
use a blue dot to mark the position of each convID that has expected=1

import pandas as pd
import matplotlib.pyplot as plt

# Read the csv file
df = pd.read_csv('data.csv')

# Separate the data based on 'expected' value
df_0 = df[df['expected'] == 0]
df_1 = df[df['expected'] == 1]

# Create the scatter plots
plt.scatter(df_0['time_to_first_token'], df_0['token_count'], color='red', label='expected=0')
plt.scatter(df_1['time_to_first_token'], df_1['token_count'], color='blue', label='expected=1')

# Label the axes
plt.xlabel('time_to_first_token')
plt.ylabel('token_count')

# Add a legend
plt.legend()

# Show the plot
plt.show()
